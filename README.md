# 📊 Ecommerce Sales Dashboard (Excel)

🚀 A dynamic and interactive **Ecommerce Sales Dashboard** built in Microsoft Excel to analyze sales performance, customer behavior, and category-wise insights.

---

## 🔍 Project Overview

This project transforms raw ecommerce data into meaningful insights using:

* 📌 Pivot Tables
* 📊 Interactive Charts
* 🎯 KPI Metrics
* 🎛️ Slicers & Filters

The dashboard helps users quickly understand trends, top-performing categories, and overall business performance.

---

## ✨ Key Features

✅ Clean and structured dataset
✅ Category-wise sales analysis
✅ Dynamic filtering using slicers
✅ KPI indicators (Total Sales, Orders, etc.)
✅ Visual insights with charts
✅ Separate sheets for raw data & dashboard

---
